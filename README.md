## Othello
혼자 오델로 게임을 할 수 있는 웹 어플리케이션
 
## Motivation
react를 사용하여 프로젝트를 해보고 싶었다. 규칙이 복잡하지 않으면서도 재미있는 Othello로 게임을 대상으로 정했다.

## Installation
git clone https://github.com/nohtaesang/othello.git
cd othello
npm install
npm start
